# Sysmon Investigation

## Objective

The objective of this investigation was to analyze Sysmon telemetry collected from the Windows 11 endpoint to understand process execution, network activity, and process termination events.

The investigation focused on the complete lifecycle of a process using Sysmon Event IDs 1, 3, and 5.

---

## Sysmon Events Investigated

| Event ID | Description |
|----------|-------------|
| 1 | Process Creation |
| 3 | Network Connection |
| 5 | Process Termination |

---

# Event ID 1 – Process Creation

## Investigation

Sysmon Event ID 1 records every newly created process.

A Notepad process was intentionally launched to generate telemetry for analysis.

The collected event included:

- Process Image
- Command Line
- Parent Process
- Parent Process ID
- Process ID
- Process GUID
- User
- Integrity Level

---

## Analyst Findings

The investigation confirmed:

- Process Name: **Notepad.exe**
- Parent Process: **explorer.exe**
- Integrity Level: **Medium**
- Process launched through normal Windows user interaction.
- Process creation was successfully ingested into Splunk.

---

# Event ID 3 – Network Connection

## Investigation

Sysmon Event ID 3 records outbound network connections initiated by processes.

During analysis, a network connection generated by Google Chrome was examined.

The event contained:

- Process Name
- Source IP Address
- Destination IP Address
- Source Port
- Destination Port
- Protocol

---

## Analyst Findings

The investigation identified:

- Process: **chrome.exe**
- Protocol: **UDP**
- Port: **5353**
- Network communication consistent with Multicast DNS (mDNS) traffic.
- No suspicious outbound communication identified.

---

# Event ID 5 – Process Termination

## Investigation

Sysmon Event ID 5 records process termination events.

The previously created Notepad process was closed to generate termination telemetry.

The investigation correlated the termination event with the earlier process creation event.

---

## Analyst Findings

The investigation confirmed:

- Process: **Notepad.exe**
- Process ID matched the original creation event.
- Process GUID matched the original process.
- Complete process lifecycle successfully reconstructed.

---

# Process Lifecycle Correlation

The investigation correlated multiple Sysmon events to reconstruct endpoint activity.

```
Process Created
(Event ID 1)

        │

        ▼

Network Activity
(Event ID 3)

        │

        ▼

Process Terminated
(Event ID 5)
```

The matching Process ID and Process GUID confirmed that the observed events belonged to the same execution lifecycle where applicable.

---

# Investigation Summary

The Sysmon investigation successfully validated endpoint telemetry ingestion into Splunk Enterprise.

The collected evidence demonstrated:

- Process execution monitoring
- Parent-child process relationships
- Network communication analysis
- Process lifecycle reconstruction
- Endpoint telemetry correlation

No malicious behavior was identified during the investigation because all activities were intentionally generated within the lab environment.
